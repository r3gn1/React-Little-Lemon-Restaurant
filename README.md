## Clone and install React-Little-Lemon-Restaurant

```shell
$ git clone https://github.com/r3gn1/React-Little-Lemon-Restaurant.git
$ cd React-Little-Lemon-Restaurant
$ npm install
```

### Launch

```
npm start
```
